## The problem with magic values

When a designer says "the primary blue is #0066cc" and a developer needs to translate that into code, something invariably gets lost. This problem multiplies as the company grows: five squads, ten repositories, and the same "primary blue" exists in eight different variations scattered across the codebase.

This is where **design tokens** come in.

## What are design tokens?

Design tokens are the atomic variables of a design system. They represent design decisions — color, typography, spacing, elevation — in a platform-agnostic way.

```json
{
  "color": {
    "primary": { "value": "#0066cc" },
    "primary-hover": { "value": "#0055aa" }
  },
  "spacing": {
    "sm": { "value": "0.75rem" },
    "md": { "value": "1.5rem" }
  }
}
```

## Why tokens and not just CSS variables?

CSS variables are an implementation. Tokens are the source of truth. The difference:

- **Tokens** exist in Figma, in JSON, in Swift, in Kotlin — on any platform
- **CSS variables** only exist in the browser
- Tokens enable **automatic synchronization** between design and code via tools like Style Dictionary

## The three levels of tokens

A well-defined taxonomy separates tokens into three layers:

### 1. Primitive tokens (Global)

These are the raw values. Never used directly in components.

```
color.blue.500 = #0066cc
color.blue.600 = #0055aa
```

### 2. Semantic tokens (Alias)

Give meaning to primitive tokens. Answer the question "what is it for?".

```
color.action = color.blue.500
color.action-hover = color.blue.600
```

### 3. Component tokens

Specific to a component. Point to semantic tokens.

```
button.primary.background = color.action
button.primary.background-hover = color.action-hover
```

## The result in practice

When we migrated to tokens in the design system, the design team was able to refactor the theme from 8 hours of work to 15 minutes — just updating the primitive tokens. Everything else cascaded automatically.

That's the beauty of well-structured tokens: **change in one place, propagate everywhere**.
